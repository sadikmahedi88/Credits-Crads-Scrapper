## 🚀 Features

<p align="center">
  <img src="https://readme-typing-svg.demolab.com?font=Fira+Code&weight=500&size=20&duration=2000&pause=500&color=00FF00&center=true&vCenter=true&width=800&lines=✅+Valid+Credit+Card+Generation;🔍+Real-Time+BIN+Lookup;📤+Auto-Send+to+Telegram;🖼️+Custom+Image+Attachment;⚡+Rate+Limiting+%26+Pause;📊+Live+Statistics+Display" alt="Features Typing" />
</p>

- ✅ **Valid Credit Card Generation** – Uses Luhn algorithm to generate valid card numbers.
- 🔍 **Real-Time BIN Lookup** – Fetches card details (bank, country, type) via BIN API.
- 📤 **Auto-Send to Telegram** – Sends generated cards directly to your Telegram channel/group.
- 🖼️ **Custom Image Attachment** – Sends each card with a custom image (Murphy.jpg).
- ⚡ **Rate Limiting & Pause** – Avoids Telegram rate limits with smart pauses.
- 📊 **Live Statistics** – Shows real-time progress and final success/failure counts.
- 🛡️ **No Proxy Required** – Simple and direct connection.

## 🛠️ Installation

### 1. Clone the Repository
```bash
git clone https://github.com/sadikmahedi88/credit-card-generator.git
cd credit-card-generator
```

2. Install Dependencies

```bash
pip install -r requirements.txt
```

3. Configure Settings

Edit config.py and add your:

· Telegram Bot Token
· Chat ID
· Custom image path

```python
BOT_TOKEN = "YOUR_BOT_TOKEN"
CHAT_ID = "YOUR_CHAT_ID"
PHOTO_PATH = "attached_assets/Murphy.jpg"
```

4. Run the Bot

```bash
python main.py
```
